    Text Analysis ARIMA Cosine Similarity Python Jupyter Notebook
    Time-series Analysis and Prediction Using ARIMA Algorithm for Text Processing.

    The project focused on deepdive into text from Indonesian news articles in 2001-2002. The dataset contained the extraction of title and content of 5449 articles, which then separated into 4 sections representing one semester of each year. The aim is for better understanding of the characteristics between articles, and the results are:

  • Measuring the identicality between articles using the Cosine Similarity approach. Later known, in this case, articles with similar content tend to have >0.12 value for the Cosine Similarity.
  • Correlation between articles length in the same month but different year is relatively high, if December was excluded.
  • Despite high correlation, the ARIMA model is unable to predict the length of articles based on historical data.
